# Urban Mobility & Economic Productivity — LATAM Cities

## Objective
Identify cities where transportation infrastructure investment would maximize economic productivity.

## Data Sources
- TomTom Traffic Index
- OECD Cities Economic Indicators

## Methodology
- Data cleaning and merging using Python (Pandas)
- Exploratory analysis and visualization
- Comparative city analysis

## Key Findings
- Bogotá and Santiago show high congestion with low GDP per capita
- Mexico City has extreme congestion with strong economic potential
- Montevideo demonstrates efficient mobility and strong productivity

## Recommendation
Prioritize investment in Bogotá and Santiago for maximum economic impact.

## Tools
Python | Pandas | Matplotlib | Data Analysis
