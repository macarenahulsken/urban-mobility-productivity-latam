## Movilidad Urbana y Productividad Económica — Ciudades LATAM

### Objetivo
Identificar ciudades donde la inversión en infraestructura de transporte maximizaría el impacto económico.

### Fuentes de datos
- TomTom Traffic Index
- OECD Cities

### Metodología
- Limpieza e integración de datos con Python (Pandas)
- Análisis exploratorio y visualizaciones
- Comparación entre congestión y productividad económica

### Hallazgos clave
- Bogotá y Santiago combinan alta congestión y bajo PIB per cápita
- Ciudad de México presenta congestión extrema con alto potencial económico
- Montevideo demuestra alta productividad con baja congestión

### Recomendación
Priorizar inversión en Bogotá y Santiago por mayor retorno económico potencial.

### Herramientas
Python | Pandas | Matplotlib | Análisis de datos
